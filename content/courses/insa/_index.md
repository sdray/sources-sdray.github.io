## Prerequisite

Install the packages `ade4` and `adegraphics` in your R session:

```r
install.packages("ade4")
install.packages("adegraphics")
```

To be able to use graphical functionalities of `adegraphics`, first load `ade4`.

```r
library(ade4)
library(adegraphics)
```
